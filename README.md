# ETHEOS — Philosophy for the Modern Soul

**ETHEOS** is a premium, cinematic one-page website for a modern philosophy, wisdom, or conscious-living brand.  
It blends ancient philosophical language with a dark luxury visual system, gold accents, editorial typography, smooth animations, and immersive interactions.

The page is built as a standalone HTML file, making it easy to preview, customize, and deploy without a complex build setup.

## 🌐 Deploy on GitHub Pages

GitHub Pages usually looks for a file named `index.html`.

To deploy this site properly:

1. Rename:

```bash
etheos-ultra.html
```

to:

```bash
index.html
```

2. Push the file to your GitHub repository.

3. Go to:

```bash
Settings → Pages
```

4. Under **Build and deployment**, choose:

```bash
Deploy from a branch
```

5. Select:

```bash
main / root
```

6. Save.

Your site will be live at:

```bash
https://your-username.github.io/your-repository-name/
```

---

## ⚠️ Important GitHub Pages Note

If GitHub Pages opens the README instead of the website, it usually means there is no `index.html` file in the root folder.

Fix it by renaming:

```bash
etheos-ultra.html
```

to:

```bash
index.html
```

Then commit and push again.

## 🎨 Customization Guide

### Change Brand Name

Search for:

```html
ETHEOS
```

Replace it with your own brand name.

### Change Tagline

Search for:

```html
Philosophy for the Modern Soul
```

Replace it with your preferred tagline.

### Change Colors

Inside the `:root` CSS block, edit variables such as:

```css
--obsidian: #0a0a0a;
--gold: #c9a962;
--ivory: #f5f2ed;
--pearl: #e8e4df;
```

### Change Fonts

The project currently uses:

- Bodoni Moda
- Cormorant Garamond
- Playfair Display
- Instrument Sans

You can change the Google Fonts import in the `<head>` section.

### Change Sections

Edit the section blocks directly inside the HTML:

```html
<section class="about" id="about">
<section class="principles" id="principles">
<section class="offerings" id="offerings">
<section class="thinkers" id="thinkers">
<section class="newsletter" id="newsletter">
```
